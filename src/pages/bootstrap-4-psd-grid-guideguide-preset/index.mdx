I created a simple boilerplate for [Bootstrap 4's Grid](http://v4-alpha.getbootstrap.com/layout/grid/#grid-options) with the awesome GuideGuide plugin in Photoshop.

The template includes artboards for: 

 * XL ( 1140px )
 * LG ( 940px )
 * MD ( 720px )
 * SM ( 576px )
 * XS ( < SM )
